# Fluxo de Navegação

1. **MainActivity (Feed)**: Ecrã de entrada com navegação vertical.
2. **Transição Feed → Detalhes**: Acionada por um toque simples na imagem ou numa miniatura da barra de favoritos.
3. **Transição Detalhes → Feed**: Através do botão de retrocesso (*Back*) ou navegação de sistema.
4. **Interações Rápidas**: O duplo toque na imagem não altera o ecrã, mas aciona a lógica de "Like" e animação visual.

# Funcionalidades do Sistema

1. **Recuperação de Dados**: Consumo assíncrono de imagens via Dog CEO API.
2. **Interface Vertical**: Navegação em ecrã total através de `ViewPager2`.
3. **Pagina Infinita**: Carregamento automático de novos conteúdos mediante a navegação do
   utilizador.
4. **Sistema de Reação (Like)**: Animação visual de coração por duplo toque ou botão
   dedicado.
5. **Gestão de Favoritos**: Implementação de uma fila FIFO (*First-In, First-out*) com limite de 5
   itens.
6. **Persistência de Dados**: Armazenamento local de favoritos e cache de até 50 itens para acesso
   offline.
7. **Transferência de Conteúdo**: Capacidade de armazenamento de imagens na galeria pública do
   dispositivo via `MediaStore`.
8. **Monitorização da Conetividade**: Deteção em tempo real do estado da rede com indicadores
   visuais.

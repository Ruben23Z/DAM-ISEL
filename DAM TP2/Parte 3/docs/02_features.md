# Features

1. Fetch dog images from Dog CEO API
2. Display images in a vertical feed (TikTok-style)
3. Infinite Scrolling (automatically loads more as you swipe)
4. Like Feature with Pop Animation (Double-tap or Button)
5. Favorites System (FIFO queue, max 5 items)
6. Download Dog Images to device gallery
7. Refresh images manually
8. Show loading indicators
9. Handle API errors gracefully

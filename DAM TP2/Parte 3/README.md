# Image Feed App (TikTok Style)

## Description
Android application that displays images from the Unsplash API in a vertical swipe feed (similar to TikTok but for images).

## Features
- Infinite vertical image feed
- Swipe down to load new images
- Refresh functionality
- MVVM architecture

## API Used
Unsplash API:
https://unsplash.com/developers

## How to Run
1. Open project in Android Studio
2. Add your Unsplash API key
3. Build and run on emulator or device
